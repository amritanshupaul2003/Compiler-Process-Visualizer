# Compiler-Process-Visualizer
Compiler Visualizer - An interactive web app that visually demonstrates compilation phases in real-time. Write code, watch it tokenize, parse into AST, and execute with beautiful visualizations. Supports Java, C, C++, C#. Build for educational purpose.
